# C.A.P.I. Website - Guía de Trabajo

### Estructura
- `/src`: Aquí es donde editamos todo. **Nunca edites la carpeta /public**.
- `/src/es`: Páginas en español.
- `/src/en`: Páginas en inglés.
- `/src/_data`: Aquí están las traducciones del menú y footer.
- `/src/assets/img`: Sube aquí todas las fotos nuevas.

### Comandos
1. `npm install` - Instalar herramientas la primera vez.
2. `npx @11ty/eleventy --serve` - Para ver los cambios en vivo en el navegador.
